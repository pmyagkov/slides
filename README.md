##Сборка
####Простая сборка
`make`
####Сборка с отслеживанием изменений
`make watch`

##Запуск
####Сервер:
Запуск без скачивания новых слайдов из диска
`node server/app`

Запуск со скачиванием
`node server/app --update`

####Клиент:
Достаточно открыть `index.html` в браузере
